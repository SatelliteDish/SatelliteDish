### What I'm working on

Not Socials is a startup building a way for people to connect through NFC. There's a physical manufactured product, a live site, and a full platform behind it waiting on a marketing push to launch. I own the technical side, which covers a Next.js web platform, a React Native mobile app, a serverless AWS backend, Stripe e-commerce, custom auth, and the infrastructure tying it all together. We got into the New Ventures BC accelerator, which is how the whole thing got properly off the ground.

### Security

I do a mix of red and blue team work on Hack The Box. The offensive side is honestly just fun, there's something satisfying about thinking through how a system can be broken that I find hard to replicate elsewhere. The defensive side keeps the other half of the picture honest, and both bleed into how I write and review code day to day.

Working towards the CPTS certification, which covers the full penetration testing lifecycle and is about as practical as security certs get. Got a voucher through an HTB Academy annual sub, so it's happening one way or another.

### How I got here

C# and Unity, making games nobody played. It's a surprisingly good foundation — game dev makes you think about performance, state, and user feedback all at once, and a lot of that carries forward. Went from there to web, then systems, then infrastructure. Rust was somewhere in the middle and became a permanent fixture.

Self-taught, which I mention not as a badge but as context for why the path looks the way it does.
